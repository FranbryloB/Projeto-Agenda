# Projeto-Agenda
# A Agenda Virtual é uma aplicação desenvolvida para organizar compromissos, tarefas e contatos de forma simples, rápida e acessível.
# O objetivo do projeto é oferecer uma solução digital intuitiva para facilitar o tarefas do dia a dia.

# Funcionalidades:
# Cadastrar compromissos (data e descrição)
# Listar tarefas
# Editar, excluir e marcar como concluído
